## Course Overview
This course focuses on the computational foundations of sequential decision-making in AI systems, with an emphasis on enabling agents to operate efficiently in complex and uncertain environments.

We cover two central paradigms:
- Automated Planning (model known)
- Reinforcement Learning (RL) (model partially known or unknown)

The course combines theoretical foundations with hands-on implementation and evaluation in practical environments.

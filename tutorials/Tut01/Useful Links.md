# Useful Links

## [Official Lambda Server Documentation](https://aihpc.cs.technion.ac.il/lambda/)
Here you will find all the official documentation for using the Lambda servers: 

## [VSCode on GPU Cluster](https://aihpc.cs.technion.ac.il/vscode-on-gpu-cluster/)
Here you will find instructions on how to connect VSCode to a GPU node for easy development and debugging.


## [SASE - Secure Access Service Edge](https://cis.technion.ac.il/en/services/sase/)

These are instructions on how to install and use the SASE client, which is required to connect to the Lambda servers from outside the Technion network.

## [SLURM Documentation](https://slurm.schedmd.com/documentation.html)
This is the official documentation website for SLURM, the workload manager used on the Lambda servers.

## [Miniforge Anaconda Repo](https://github.com/conda-forge/miniforge)
This is the official GitHub repository for the open-source Anaconda alternative Miniforge.
